# 🔐 Как войти в админ панель

## 📋 Требования для доступа:

1. ✅ Вы должны быть **авторизованы** (войти в систему)
2. ✅ Ваш аккаунт должен иметь роль **ADMIN** или **MODERATOR**

---

## 🚀 Способ 1: Через Supabase Dashboard (Рекомендуется)

### Шаг 1: Найдите ваш email в базе данных

1. Откройте **Supabase Dashboard** → ваш проект → **Table Editor**
2. Найдите таблицу **User**
3. Найдите вашу запись по email (который вы использовали для входа через Google или Email/Password)

### Шаг 2: Измените роль на ADMIN

1. Откройте вашу запись в таблице **User**
2. Найдите поле **role**
3. Измените значение с `USER` на `ADMIN`
4. Нажмите **Save**

### Шаг 3: Войдите в админ панель

1. Откройте ваш сайт: `https://namelonlar.vercel.app`
2. Убедитесь, что вы авторизованы (видите иконку профиля)
3. Перейдите по адресу: `https://namelonlar.vercel.app/admin`
4. Должна открыться админ панель! ✅

---

## 🚀 Способ 2: Через SQL запрос в Supabase

### Шаг 1: Откройте SQL Editor

1. Supabase Dashboard → ваш проект → **SQL Editor**
2. Нажмите **New Query**

### Шаг 2: Выполните SQL запрос

Замените `your-email@example.com` на ваш реальный email:

```sql
UPDATE "User" 
SET role = 'ADMIN' 
WHERE email = 'your-email@example.com';
```

Нажмите **Run** или `Ctrl+Enter`

### Шаг 3: Войдите в админ панель

1. Откройте ваш сайт
2. Убедитесь, что вы авторизованы
3. Перейдите на `/admin`

---

## 🚀 Способ 3: Через Prisma Studio (Локально)

Если вы работаете локально:

### Шаг 1: Запустите Prisma Studio

```bash
npm run db:studio
```

### Шаг 2: Найдите и отредактируйте пользователя

1. Откройте браузер на `http://localhost:5555`
2. Найдите таблицу **User**
3. Найдите вашу запись
4. Измените поле **role** на `ADMIN`
5. Сохраните изменения

---

## 🔍 Проверка доступа

После изменения роли:

1. **Выйдите** из системы (если вы уже авторизованы)
2. **Войдите снова** (чтобы обновить сессию)
3. Перейдите на `/admin`

Если вы видите админ панель с меню:
- ✅ Boshqaruv paneli
- ✅ E'lonlar
- ✅ Foydalanuvchilar
- ✅ Kategoriyalar
- ✅ Bannerlar
- ✅ Sozlamalar

Значит все работает! 🎉

---

## ⚠️ Важно:

- Роль `ADMIN` дает полный доступ ко всем функциям
- Роль `MODERATOR` дает доступ к модерации контента
- Роль `USER` - обычный пользователь (нет доступа к админ панели)

---

## 🐛 Если не работает:

1. **Проверьте, что вы авторизованы:**
   - Должна быть видна иконка профиля вместо кнопки "Kirish"

2. **Проверьте роль в базе данных:**
   - Убедитесь, что роль действительно изменилась на `ADMIN`

3. **Выйдите и войдите снова:**
   - Сессия может кешировать старую роль

4. **Проверьте URL:**
   - Должен быть: `https://namelonlar.vercel.app/admin`
   - Не `/admin/` (со слешем в конце)

---

## 📝 Быстрая команда SQL для Supabase:

Если вы знаете свой email, выполните этот запрос в SQL Editor:

```sql
-- Замените на ваш email
UPDATE "User" 
SET role = 'ADMIN' 
WHERE email = 'ваш-email@gmail.com';

-- Проверьте результат
SELECT id, email, name, role 
FROM "User" 
WHERE email = 'ваш-email@gmail.com';
```
