## ❌ Проблема: NextAuth возвращает 401

Ошибка `CredentialsSignin` означает, что:
1. Пароль не совпадает с тем, что в базе данных
2. Или есть проблема с проверкой пароля

## 🔍 Решение: Проверьте пароль

### Вариант 1: Вы помните пароль

Если вы помните пароль, который устанавливали:
1. Попробуйте войти снова
2. Убедитесь, что:
   - Email: `esalimov15@gmail.com`
   - Пароль вводите точно так, как устанавливали
   - Нет лишних пробелов

### Вариант 2: Пароль забыт или не помните

Если пароль забыт, нужно установить новый:

#### Шаг 1: Сгенерируйте новый bcrypt hash

Используйте онлайн генератор или Node.js:

```bash
node -e "const bcrypt = require('bcryptjs'); bcrypt.hash('ВАШ_НОВЫЙ_ПАРОЛЬ', 10).then(hash => console.log(hash));"
```

Или используйте онлайн: https://bcrypt-generator.com/

#### Шаг 2: Обновите пароль в базе данных

Выполните SQL в Supabase:

```sql
UPDATE "User" 
SET password = 'ВАШ_BCRYPT_HASH' 
WHERE email = 'esalimov15@gmail.com';
```

#### Шаг 3: Попробуйте войти

Войдите с новым паролем.

## 🆘 Если ничего не помогает

Проверьте логи Vercel:
1. Vercel Dashboard → ваш проект → **Deployments**
2. Найдите последний deployment
3. Перейдите в **Functions** или **Logs**
4. Найдите логи с `[AUTH]` - там будет детальная информация

Логи покажут:
- Найден ли пользователь
- Правильный ли формат пароля
- Совпадает ли пароль
